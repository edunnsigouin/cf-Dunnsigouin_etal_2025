To run calc-fss-forecast-daily-anomaly.py you have to run the following scripts for each data set.
and for the correct forecast filenames and/or dates.Starting point is after having downloaded the data.

era5:
1) calc-daily-from-6hourly-era5.py
2) calc-daily-to-forecast-fromat-era5.py
3) calc-clim-to-forecast-format-era5.py
4) calc-forecast-to-anomaly-era5.py

s2s:
1) calc-accumulated-variables-6hourly-s2s.py
2) calc-daily-from-6hourly-s2s.py
3) calc-clim-hindcast-s2s.py
4) calc-forecast-to-anomaly-s2s.py
